Configurar Firebase (Gratuito)

Para que os votos sincronizem entre celulares em tempo real, siga estes passos simples:

  1. Acesse console.firebase.google.com
  2. Crie um projeto (nome: aldeia-wine)
  3. Clique em Realtime Database → Criar banco de dados
  4. Escolha o modo teste (30 dias gratuito)
  5. Em Configurações do Projeto → Adicionar app Web
  6. Copie o objeto firebaseConfig
  7. Abra este arquivo HTML num editor de texto e cole os valores no início do script
